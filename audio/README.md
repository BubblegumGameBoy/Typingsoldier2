# BGM音楽ファイル

ゲームはこのフォルダ内の以下のファイル名でmp3を参照します。
Google Driveの元ファイルを、下記の名前にリネームしてこの `audio/` フォルダにアップロードしてください。

| ゲーム内の用途 | このフォルダに置くファイル名 | Google Driveの元ファイル |
|---|---|---|
| Wave1 通常 | `wave1.mp3` | wave1.mp3 |
| Wave2〜5 通常 | `normal.mp3` | Wave2-4 - Normal song.mp3 |
| Wave2 ボス戦 | `wave2_boss.mp3` | Wave2 - BOSS FIGHT.mp3 |
| Wave3 ボス戦 | `wave3_boss.mp3` | Wave3 - Boss fight.mp3 |
| Wave5 ボス戦 | `wave5_boss.mp3` | Wave5 -Boss fight.mp3 |
| 魔王登場（Wave4ボス戦） | `demon.mp3` | Overload_Ratio_3.mp3 |
| 勇者登場（Wave4） | `hero.mp3` | Wave4 - 勇者誕生.mp3 |

※ ファイルが無くてもゲームは動作します（その曲が鳴らないだけ）。
